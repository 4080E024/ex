# For 迴圈(loop)
```
for name in ['orange', 'apple']:
	print(name)
  答案是:orange apple

```

```
for index in range(0, 6):
	print(index)
答案是:
0
1
2
3
4
5
```
### 程式設計題
```
使用For 迴圈(loop)計算1+2+3.......+101
sum=0

for x in range(1,101):
  sum +=x
  
print(sum)  
答案是:5050
```

```
使用For 迴圈(loop)計算sum=0

for x in range(1,101,2):
  sum +=x
  
print(sum)  
答案是:2500
```
