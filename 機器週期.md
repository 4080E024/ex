# 機器週期
```
機器週期分成：擷取 解碼 執行 儲存 這四項
```
# 擷取
```
控制單元會要求命令系統複製下一項指令到 CPU 的指令暫存器中，而將要被複製的指令位址會存在程式計數器當中。
```
# 解碼
```
當指令被擷取到指令暫存器後，會由控制單元加以解碼執行。
```
# 執行
```
在指令解碼完成後，控制單元會送出工作命令到 CPU 中的組成元件當中。
```
# 儲存
```
送至記憶單元，儲存結果
```
